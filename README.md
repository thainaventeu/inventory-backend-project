   ```bash

3. **Execute the server in background** 
    ```bash
    go run main.go &

4. **Set up the database:**
Create a local PostgreSQL database named inventory.
Update the connection settings in the config/database.go file:

```bash
    const (
    Host     = "localhost"
    Port     = 5432
    User     = "your_username"
    Password = "your_password"
    DBName   = "inventory"
)
